# makemore

a reimplementation of [@karpathy](https://github.com/karpathy)'s [makemore](https://github.com/karpathy/makemore) — a tiny character-level language model, built to learn NLP from the ground up.

> *makes more of whatever you give it.*

## what it does

takes a list of words and trains a model to generate new ones, character by character. start simple (bigrams), go deep (MLPs, RNNs, Transformers).

## why

learning by doing. no better way to understand language models than building one from scratch.
